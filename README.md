
# cyclingpower

<!-- badges: start -->
<!-- badges: end -->

The goal of cyclingpower is to compare power files from multiple power meters in R.

## Installation

You can install cyclingpower with:

``` r
remotes::install_github("jzuccollo/cyclingpower")
```
